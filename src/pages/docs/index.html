Documentation

Welcome to the FreeFSM documentation. FreeFSM is a self-hosted, open-source field service management platform for FreeBSD and Linux. Navigate through the sections below to learn how to install, configure, and use the system.

Getting Started

New to FreeFSM? Getting Started walks you through prerequisites, installation, database setup, and your first login.

Configuration

The Configuration Reference documents every environment variable FreeFSM supports, along with defaults and descriptions.

User Guide

Learn how to use each module in the User Guide — from customers and jobs to estimates, invoices, and the schedule calendar.

Deployment

Ready to deploy to production? The Deployment Guide covers systemd on Linux, rc.d on FreeBSD, and the Makefile install targets.

Development

Contributors and developers should read the Development Guide for project structure, build commands, and the entity creation workflow.

Architecture

Deep dive into the Architecture — tech stack rationale, database design decisions, and key patterns like polymorphic relations and the status workflow system.

Roadmap

See where FreeFSM is headed in the Roadmap, covering all phases from foundation through advanced integrations.

API Reference

Build integrations with the REST API, which covers 113 endpoints across 26 resource groups with standardized pagination, filtering, and eager loading.
