API Reference

FreeFSM exposes a REST API with 113 endpoints across 26 resource groups (planned for Phase 4). All endpoints use x-api-key header authentication.

Standard Query Parameters

Parameter Description
limitResults per page
pagePage number
searchFull-text search query
filter[]Array of field filters
sort[]Array of sort fields with direction
rel[]Eager loading for related resources

Webhook Events

The webhook system (Phase 4) will fire events for:

  • Job Created / Custom Status Update / Start Time Update / End Time Update
  • Estimate Created / Custom Status Update / Workflow Status Update
  • Invoice Created / Custom Status Update / Workflow Status Update

Polymorphic relations use object_type + object_id throughout, and all request/response bodies use JSON.
