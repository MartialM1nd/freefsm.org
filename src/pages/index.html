Open-source field service
management for FreeBSD & Linux

FreeFSM is a self-hosted, open-source field service management platform. One static Go binary, PostgreSQL backend, zero npm dependencies. Manage customers, jobs, schedules, estimates, and invoices — all from your own infrastructure.

Get started in minutes

FreeBSD or Linux, PostgreSQL, and one binary. That’s it.

Terminal bash 
# Clone, configure, and run
git clone https://github.com/MartialM1nd/freefsm.git
cd freefsm
cp .env.example .env

# Edit .env with your DB credentials, then:
make run
# → http://localhost:3000

Deploy anywhere

Single static binary, ready for production on both platforms.

Linux — systemd

make install-linux
systemctl start freefsm

FreeBSD — rc.d

make install-freebsd
service freefsm start
