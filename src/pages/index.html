Open-source field service
management for FreeBSD & Linux

FreeFSM is a self-hosted, open-source field service management platform. One static Go binary, PostgreSQL backend, zero npm dependencies. Manage customers, jobs, assets, projects, timesheets, estimates, invoices, file attachments, audit logs, and more — all from your own infrastructure.

Everything you need to run your field service

From dispatch to invoicing, FreeFSM covers the full workflow with a clean, fast interface.

Dashboard

KPI cards, calendar, quick actions, and global search — see your business at a glance.

Customers

Full CRUD with search, status filtering, HTMX pagination, locations, and multi-contact support.

Jobs & Work Orders

Configurable status workflows, scheduling, arrival windows, multi-visit support, subtasks, and team assignments.

Schedule

Calendar and list views with clickable job cards, drag-and-drop status changes, and filter by team or user.

Estimates

Line items editor, tax calculation, PDF generation, and convert to invoice on approval.

Invoices

Line items from pricebook, payment recording, status workflow, and PDF generation.

Items / Pricebook

Service and product catalog with SKU tracking, pricing tiers, and tax settings.

Assets

Equipment tracking with types, statuses, manufacturer, model, serial numbers, and service history.

Projects

Group jobs under a customer with progress tracking, nested work orders, and status overview.

Timesheets

Clock in/out per user with GPS coordinates, manual entry, and job-linked time tracking.

Tags & Custom Fields

Color-coded labels and user-defined fields on any object — customers, jobs, assets, invoices, and more.

Comments

Threaded notes on any object with timestamps and authorship tracking.

User Management

Role-based access (admin, tech, dispatcher), welcome emails, password policies, and force password change.

Company Settings

Branding, email config, timezone, document numbering, security policies, and dark mode toggle.

Auth & Security

Setup token for initial admin, bcrypt password hashing, HTTP-only session cookies, and CSRF protection.

Activity / Audit Log

Every mutation tracked across all entities with per-entity activity widgets, admin-only admin log, and recent activity panels on every list page.

File Attachments

Polymorphic uploads on customers, jobs, estimates, invoices, and assets with MIME whitelist, inline preview for images/PDFs, and disk-based storage.

Soft-Delete / Archive

Business entities archived instead of hard-deleted with admin restore from the show page banner — nothing is truly lost.

Dependency Protection

Prevents deletion of configuration items like tags, asset types, and statuses when they are still referenced by other records.

Tech Stack

Built with modern, battle-tested technologies. Zero proprietary dependencies.

Get started in minutes

FreeBSD or Linux, PostgreSQL, and one binary. That’s it.

Terminal bash 
# Clone, configure, and run
git clone https://github.com/MartialM1nd/freefsm.git
cd freefsm
cp .env.example .env

# Edit .env with your DB credentials, then:
make run
# → http://localhost:3000

Deploy anywhere

Single static binary, ready for production on both platforms.

Linux — systemd

make install-linux
systemctl enable --now freefsm

FreeBSD — rc.d

gmake install-freebsd
service freefsm start
